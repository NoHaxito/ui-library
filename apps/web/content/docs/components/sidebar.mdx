## Usage

<hr className="mt-6 dark:border-neutral-800" />

```tsx
import {
  Sidebar,
  SidebarItems,
  SidebarItemGroup,
  SidebarItem,
} from "@sihaxito/deluxe-ui";
```

```tsx showLineNumbers
<Sidebar>
  <SidebarItems>
    <SidebarItemGroup>
      <SidebarItem></SidebarItem>
      <SidebarItem></SidebarItem>
    </SidebarItemGroup>
  </SidebarItems>
</Sidebar>
```

## Controlled state

<hr className="mt-6 dark:border-neutral-800" />

### Collapsible

You can pass <Code2>collapsed</Code2> and <Code2>onCollapsedChange</Code2> to the <Code2>Sidebar</Code2> component to handle manually the collapsed state. You can use a React global context or a useState.

<ComponentPreview name="sidebar/controlled-collapsible" />

### Open

Similar to collapsible, you can pass <Code2>open</Code2> and <Code2>onOpenChange</Code2> to the <Code2>Sidebar</Code2> component to handle manually the open state. You can use a React global context or a useState.

<ComponentPreview name="sidebar/controlled-open" />
