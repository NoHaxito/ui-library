### Tailwind CSS Setup

Deluxe UI requires Tailwind CSS to be used. If you don't have it installed, you can follow the instructions in the <a className="text-blue-600 dark:text-blue-400 hover:underline" href="https://tailwindcss.com/docs/guides/nextjs" target="_blank" rel="reopenner">official Tailwind CSS installation guide</a>.

Once you have installed Tailwind CSS you will need to update your <Code2>tailwind.config</Code2> file.

```tsx {11} showLineNumbers title="tailwind.config.js"
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",

    // Or if using 'src' directory:
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  darkMode: ["class"],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

### Import Deluxe UI styles

<Tabs defaultValue="app_router" variant="underline">
  <TabsList>
    <TabsTrigger value="app_router">App router</TabsTrigger>
    <TabsTrigger value="pages_router">Pages router</TabsTrigger>

  <TabsContent value="app_router">
  ```tsx {3} showLineNumbers title="layout.jsx"
import { Inter } from "next/font/google";
import "./globals.css";
import "@sihaxito/deluxe-ui/styles.css";

const inter = Inter({ subsets: ["latin"] });

export const metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  );
}
```
</TabsContent>
  <TabsContent value="pages_router">
  ```jsx {2} showLineNumbers title="_app.jsx"
  import '@/styles/globals.css'
  import '@sihaxito/deluxe-ui/styles.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
  ```
  </TabsContent>
</Tabs>

### Trying Deluxe UI

Now that the installation and configuration of Deluxe UI has been successfully completed, you are ready to use the components in your application.

```tsx {1,4} showLineNumbers
import { Badge } from "@sihaxito/deluxe-ui";

export function Example() {
  return <Badge>Deluxe UI</Badge>;
}
```
