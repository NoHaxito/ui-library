Deluxe UI components are built with the 'use client' directive, allowing for both server-side and client-side usage, and compatibility with both the App Router and Pages Router.

<Steps>

### Install Deluxe UI


<CodeHighlighter language="bash" code={`npm install @sihaxito/deluxe-ui`} />

### Tailwind CSS Setup

Deluxe UI requires Tailwind CSS to be used. If you don't have it installed, you can follow the instructions in the <a className="text-blue-600 dark:text-blue-400 hover:underline" href="https://tailwindcss.com/docs/guides/nextjs" target="_blank" rel="reopenner">official Tailwind CSS installation guide</a>.

Once you have installed Tailwind CSS you will need to update your `tailwind.config` file.

<CodeHighlighter
  showLines
  language="js"
  highligthLines={[11]}
  code={`/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",
 
    // Or if using 'src' directory:
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  darkMode: ["class"],
  theme: {
    extend: {},
  },
  plugins: [],
}`}
/>

### Import Deluxe UI styles

<CodeHighlighter
  showLines
  language="tsx"
  highligthLines={[3]}
  code={`import { Inter } from 'next/font/google'
import './globals.css'
import '@sihaxito/deluxe-ui/styles.css'

const inter = Inter({ subsets: ['latin'] })

export const metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  )
}`}
/>
### Trying Deluxe UI

Now that the installation and configuration of Deluxe UI has been successfully completed, you are ready to use the components in your application.

<CodeHighlighter
  showLines
  highligthLines={[1,4]}
  language="tsx"
  code={`import { Badge } from "@sihaxito/deluxe-ui";

export function Example() {
  return <Badge>Deluxe UI</Badge>
}`}
/>
